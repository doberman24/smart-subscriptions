# Smart Subscriptions

Fullstack web application for managing subscription-based services.

The project is focused on building a modern frontend with a structured backend API,
as well as practicing real-world interaction between client and server.

## Key features
- Subscription management (create, update, delete)
- REST API interaction
- Modular frontend architecture
- Backend business logic separation
- Real project-oriented structure

## Tech stack
**Frontend**
- JavaScript
- React
- REST API

**Backend**
- Node.js
- Express
- MongoDB

## Project structure

The project is split into separate repositories to preserve independent development
and commit history for frontend and backend parts:

- Frontend repository:  
  https://github.com/doberman24/smart-subscriptions-frontend

- Backend repository:  
  https://github.com/doberman24/smart-subscriptions-backend

This approach reflects a real-world fullstack development workflow.

## Project status
The project is under active development and is being used as a production-oriented
pet project.
